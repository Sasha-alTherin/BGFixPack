I. Notes

This mod will only work with Baldur's Gate, with or without Tales of The Sword Coast.

If you play without the official expansion Tales of The Sword Coast, and with unofficial expansions, you may have difficulties late in the game (enemies become unkillable, movies keep repeating, etc.). This is often because the game stores many variables, called GLOBALs, and the game without the Tales of The Sword Coast is very limited. The designers did not anticipate the many fan mods adding to the game. If necessary, use a saved-game utility such as Sword Coast Keeper or Near Infinity to remove some of the variables. Most SPRITE_IS_DEAD variables and those set early in the game are generally safe to remove, though if you do this have another saved game near the one you are modifying game-wise, since you may delete a variable you may still need.

The core component is built to be flexible. It is a design feature from development and testing. Every effort, however, has been made to ensure that you, the end user, need not modify any files. Should a situation arise and you have been asked to uninstall and modify your selection of core components, there are instructions on how to do so in the file modulecontroller.txt. It can be found in the main BG1 Fixpack directory.

II. Overview

The game Baldur's Gate and the expansion pack Tales of The Sword Coast while an enjoyable experience is known to still contain errors throughout the game. While some situations pass by unnoticeably due to specific role-play, other situations are blatantly present. There have been several attempts over the years to provide corrections to many of these situations. First there was Kevin Dorner and his fix pack called "Baldurdash". Then there was Dudley and his broad collection of fixes and arguably extreme modifications towards a more D&D Pen & Paper style experience which collectively have become known as "Dudleyville". Following the advent of WeiDU, Idobek at Gibberlings3.net began to create a weidu based fix pack which was to combine the many elements of Baldurdash and Dudleyville into a singular easy to install and uninstall package. This project was never finished and was in fact abandoned with many elements transferring to a new project called Unfinished Business. It is here that an upstart modder with barely any knowledge of the previous fix packs took on the daunting task of creating a singular weidu based fix pack.

That said, the purpose of this mod is to address a myriad of errors, glitches, crashes and other issues that hamper the enjoyment of the game.

III. Compatability

This mod is designed to work with Baldur's Gate with or without the expansion Tales of the Sword Coast. It is intended to be installed first and foremost. Every effort has been made to ensure that the mod Unfinished Business remains compatible. The fix pack should also be compatible with most other modifications. However, Baldur's Gate has been out for a long time and there are numerous older modifications out there which over write files rather than patching them. These modifications need to be dealt with carefully. General rule of thumb is for non-patching modifications to be installed first, followed by the fix pack and other patching modifications.

Should any incompatibilities be discovered, they will be listed here.

IV. Installation

Windows
The BG1 Fixpack is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have a BGFixpack folder and Setup-BGFixpack.exe in your BG folder. To install, simply double-click Setup-BGFixpack.exe and follow the instructions on screen.

Please run Setup-BGFixpack.exe in your BG folder to reinstall, uninstall or otherwise change components.

provide mac info IF a mac version gets made.

V. Components

BG1 Fixpack has but a few components

Required for all components

This component serves two purposes:

  • A singular component to uninstall and force un-installation of all other components (aside from any included optional components)
  • Establish a common base across various game installations prior to installing the core component. Most notable are:
    • The Mac version of the 5512 patch was incomplete. Files will be brought up to patch level before continuing with the core component.
    • Rare cases of installations on Windows XP having override folder data merged rather than replaced after applying the expansion Tales of the Sword Coast and the official patches, the extra files are sought and if all of them are present they are moved out of the override and into the BG1 Fixpack's backup folder.
    • There is an unused and corrupted file (ar0110.wed) which is only present in the Original Saga version of Baldur's Gate. It has been known to cause issues with other modifications and various utilities. It is replaced by the repaired version used by Unfinished Business for the Ice Island restoration. Note: This does not mean that Ice Island is restored. The file still remains unused but now in a repaired state.
